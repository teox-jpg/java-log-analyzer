# Java Log Analyzer

Eine in Java entwickelte Konsolenanwendung zur Analyse von Logdateien.

## Funktionen

- Einlesen und Verarbeiten von Logdateien
- Umwandlung der Logeinträge in Java-Objekte
- Auswertung der Log-Level (INFO, WARNING, ERROR, TRACE)
- Statistische Zusammenfassung der Logdaten
- Strukturierte Ausgabe der Ergebnisse in der Konsole

## Verwendete Technologien

- Java
- Objektorientierte Programmierung (OOP)
- Collections
